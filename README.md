# SA Trainer — Тренажёр системного аналитика

Интерактивный учебный проект: теория системного анализа, базы данных, SQL-практика на PostgreSQL.

## Что внутри

- **28+ статей** по системному анализу, API, БД и интеграциям
- **6 SQL**, **7 API**, **4 Swagger**, **4 BPMN** заданий
- **3 кейса** проектирования БД
- **SQL-песочница**, **API-песочница**, **Swagger UI**
- Учебная схема **TechStore** (интернет-магазин)

## Быстрый старт

```bash
npm install
cp .env.example .env
npm run db:init
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000)

## Подключение к БД

Параметры в `.env`:

| Переменная | Описание |
|------------|----------|
| `DB_HOST` | Хост PostgreSQL |
| `DB_PORT` | Порт |
| `DB_NAME` | Имя базы |
| `DB_USER` | Пользователь |
| `DB_PASSWORD` | Пароль |
| `SQL_READONLY` | `true` — только SELECT в песочнице |

## Структура проекта

```
content/          # Теория (Markdown)
sql/              # Схема и seed-данные
src/app/          # Next.js страницы и API
src/components/   # UI-компоненты
src/lib/          # БД, контент, упражнения
scripts/          # init-db.mjs
```

## Команды

| Команда | Описание |
|---------|----------|
| `npm run dev` | Запуск в режиме разработки |
| `npm run db:init` | Создать схему и данные |
| `npm run db:reset` | Удалить таблицы и пересоздать |

## Рекомендуемый порядок обучения

1. Системный анализ → роль, требования, user stories, BPMN
2. API и HTTP → теория → API-песочница → Swagger UI
3. Базы данных → ER, нормализация, SQL-практика
4. Практика → Swagger, BPMN-кейсы, проектирование БД
5. Интеграции и API-контракты
